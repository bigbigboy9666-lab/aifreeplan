Tencent Marvis

★★★★☆ 10M tokens/day Paid from: Completely free

Visit official site →

Operating system-level AI assistant by Tencent's App Store team. Fully launched May 21. 10M free tokens daily, powered by DeepSeek V4 and Hunyuan models. Windows, Mac, Android. Privacy mode (local model, zero upload) and efficiency mode (cloud-edge). 8 built-in agents for file search, system settings, remote control, document processing.

Free Credits Details

  • Amount: 10M tokens
  • Reset: Resets daily
  • Watermark: None
  • Commercial: Personal use

Key Features

  • OS-level AI Agent
  • Local file smart search
  • One-sentence PC settings
  • Phone remote control PC
  • Deep file understanding
  • Privacy mode (local model)
  • 8 built-in specialized agents
  • Windows/Mac/Android

Pros

  • Completely free, 10M tokens daily
  • OS-level agent, truly controls PC
  • Privacy mode, zero file upload
  • Phone remote control of PC
  • 8 built-in agents, ready to use
  • DeepSeek V4 + Hunyuan dual models

Cons

  • Requires 16GB RAM minimum
  • Mac needs M1 or newer
  • iOS not yet available (mid-June)
  • Tencent ecosystem lock-in risk

Tips

  • Choose mode on setup: Privacy for sensitive files, Efficiency for speed
  • Install Marvis on phone + scan QR to pair, enables remote PC control
  • 10M tokens = ~5000 conversations, more than enough for daily use
  • Test privacy mode with non-sensitive files first before processing important docs
