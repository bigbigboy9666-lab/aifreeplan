Setup Steps

  1. Visit maas.xfyun.cn
  2. Find Qwen3.6-35B-A3B or Qwen3.5-35B-A3B in the model marketplace
  3. Click "API Call", create an authorized app, get modelId and APIKey
  4. Download CC Switch (open source)
  5. Add provider with API Key:
    • Claude Code endpoint: https://maas-api.cn-huabei-1.xf-yun.com/anthropic
    • Codex endpoint: https://maas-api.cn-huabei-1.xf-yun.com/v2
  6. Enter modelId and model name, test and enable
