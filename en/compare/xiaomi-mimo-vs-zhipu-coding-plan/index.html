Xiaomi MiMo Code

Xiaomi's AI coding agent forked from OpenCode, open-sourced under MIT. The killer feature is the built-in MiMo Auto channel — zero config, free for 1 month, runs MiMo-V2.5 with 1M token context. Coding performance is on par with Claude Sonnet 4.6 on SWE-Bench, but 40-60% more token-efficient. Highlights: cross-session memory, voice input, auto knowledge distillation. After the free period, you need a Token Plan subscription.
