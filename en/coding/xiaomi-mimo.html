Home › LLM › Xiaomi MiMo Free Credits Details

Xiaomi made a surprising move this year — they forked OpenCode and built MiMo Code, an AI coding agent released under the MIT license. Honestly, I didn't expect much at first (plenty of big companies have tried this), but after using it, there's real substance here.

The biggest selling point is dead simple: install it and start coding for free. Pick the MiMo Auto channel, zero configuration needed, no API key registration, running MiMo-V2.5 with a 1M token context window, free for 1 month. For anyone curious about AI-assisted coding but not ready to pay, the barrier to entry is basically zero.

Can it actually compete? The numbers from Xiaomi show MiMo Code with V2.5-Pro matching Claude Opus 4.6 and GPT-5.4 on benchmarks like SWE-Bench Pro. That's the Pro version though — the free channel runs the standard V2.5, which is still solid for everyday coding. The real standout is token efficiency: 40-60% less consumption than Claude for the same tasks, meaning your budget goes significantly further.

What sets MiMo Code apart from other coding tools? Cross-session persistent memory stored in SQLite — it remembers what you were working on last time without you re-explaining. Voice input via /voice lets you dictate code using TenVAD plus Xiaomi's in-house ASR. The /dream command auto-extracts useful knowledge from your session history into project memory, like the AI taking its own notes.

MiMo Code Free Credits Details

  • MiMo Auto Channel: Free for 1 month, MiMo-V2.5 model, 1M context, zero config
  • Platform Signup Bonus: Free credits on registration, valid 40 days
  • Referral Rewards: $2 trial credits for both parties, up to 20 invites ($40 max)
  • Night Discount: 0.8x consumption during off-peak (00:00-08:00 Beijing Time)
  • Free Models: MiMo-V2.5 (free channel) / V2.5-Pro requires payment
  • TTS Models: All free for a limited time

Paid Plans

  • MiMo Auto: Free for 1 month (then paid)
  • Token Plan Lite: 4.1B Credits/month, good for individual devs
  • Token Plan Standard/Pro/Max: More Credits for heavy usage
  • Pay-as-you-go: V2.5 $0.14/M input, $0.28/M output; Pro $0.435/M input, $0.87/M output

Key Features

  • MiMo Auto zero-config free channel
  • Cross-session persistent memory (SQLite FTS5)
  • Compose mode for spec-driven development
  • Voice input (/voice command)
  • /dream auto knowledge extraction + /distill skill mining
  • Goal stop-condition verification (prevents premature stops)
  • Subagent parallel system
  • 1M token context window
  • MIT open source, BYOK supported

Pros

  • Free for 1 month with zero setup
  • Coding performance on par with Claude Sonnet 4.6

  • Cross-session memory actually works well

Tips
