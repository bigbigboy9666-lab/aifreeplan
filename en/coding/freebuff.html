Freebuff

★★★★☆ Free, ads shown in CLI Paid from: Codebuff from $100/mo

Visit official site →

Free AI coding agent that edits code via natural language in terminal. Based on paid Codebuff, ad-supported. Uses DeepSeek V4 Flash by default, supports Kimi K2.6, MiniMax M2.7 and other models.

Freebuff Free Credits Details

  • Amount: Free, ads shown in CLI

Freebuff Key Features

  • Natural language coding in terminal
  • 9 built-in sub-agents
  • DeepSeek V4 Flash
  • Code review
  • Browser use
  • Deep thinking
  • Open source MIT license

Pros

  • Completely free, ads for credits
  • No config needed, works out of box
  • Supports multiple domestic models
  • Built-in code review and browser use
  • Open source MIT license

Cons

  • Ads in terminal
  • DeepSeek V4 Pro trains on data
  • Not available in all countries
  • Requires internet, no offline use

Tips

  • Use V4 Flash not V4 Pro for company code
  • Use @filename to reference specific files
  • /init to generate project knowledge file
  • Connect ChatGPT subscription for deep thinking
