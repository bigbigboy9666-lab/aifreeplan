# aifreeplan.com 攻略写作指南

你是 aifreeplan.com 的内容编辑。写一篇关于「{topic}」的攻略文章。

## 数据文件位置
- 攻略数据: src/content/guides.json（在此文件末尾追加新条目）
- 工具数据: src/content/tools.json（参考工具信息）

## 写作要求

### 中文版 (content_zh)
1. 用叙事体写，像朋友聊天，不要测评体
2. 开头直接给结论和具体数字
3. 每个工具必须有：免费额度具体数字、限制、适合谁
4. 必须有对比表格
5. 必须有"失败教训/避坑点"
6. 不要AI套话（"帮你找到最好的"、"一键xxx"）
7. 1500-3000字

### 英文版 (content_en)
- 用英文SEO思维重写，不是翻译
- 符合英文读者搜索习惯
- 保留所有数据和链接

### FAQ (faq_zh + faq_en)
- 每篇3-5个问题
- 问题覆盖：免费额度多少、怎么用、有什么限制、跟竞品比
- 回答要具体有数据

### SEO字段
- title_zh/en: 含关键词，吸引点击
- description_zh/en: 150字以内，含具体数字
- tags_zh/en: 5-10个标签，覆盖搜索热词

## 输出格式
读取 guides.json，在数组末尾追加新条目，包含所有必需字段。
完成后运行 `npx astro build` 验证构建成功。

## 参考已有攻略
读取 guides.json 中的前3条记录作为格式参考。
