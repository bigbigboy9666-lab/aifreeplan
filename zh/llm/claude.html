Claude

★★★★☆ Sonnet 模型有限次/天 付费起步: $20/month

访问官网 →

Anthropic 出品，以安全性和长文本处理著称。免费版每天有消息条数限制，Pro 用户可用 Claude Code 做终端编程。代码能力在多个基准测试里排前列。

Claude 免费额度详情

  • 额度: Sonnet 模型有限次
  • 重置: 每日重置
  • 水印: 无
  • 商用: 免费版可商用

Claude 主要功能

  • 200K 超长上下文
  • Claude Code 命令行编程
  • 扩展思考（Extended Thinking）
  • Artifacts 代码/文档预览
  • Projects 项目管理
  • 文件上传分析
  • 多语言对话

优点

  • 200K 上下文窗口，同类最长
  • 编程能力在评测里排名前列
  • 回答逻辑清晰，幻觉少
  • Claude Code 命令行编程体验好
  • 免费版 Sonnet 日常够用

缺点

  • 免费版每天用量有限
  • Opus 模型和扩展思考需要 Pro
  • 国内访问需要特殊网络
  • 没有图片生成功能

使用建议

  • 长文档分析首选 Claude，200K 上下文不是盖的
  • 用 Projects 功能给不同项目建独立空间
  • Claude Code 是免费的命令行编程工具，比 ChatGPT 更适合写代码
