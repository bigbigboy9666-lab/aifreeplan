讯飞星辰 MaaS 免费额度详情

科大讯飞 | AI大模型 | ★★★★☆

科大讯飞推出的一站式大模型服务平台，聚合了讯飞星火Spark、DeepSeek、Qwen等主流模型。目前活动期间可免费不限量使用Qwen3.6-35B-A3B和Qwen3.5-35B-A3B，活动持续到6月底，无限Token。支持API调用，可通过CC Switch接入Claude Code、Codex等工具使用。

免费额度详情

  • 额度: 活动期间不限量（到6月底）
  • 重置: 无限Token
  • 模型: Qwen3.6-35B-A3B、Qwen3.5-35B-A3B
  • 上下文: 128K窗口
  • 水印: 无
  • 商用: 需确认活动条款

主要功能

  • Qwen3.6/3.5-35B免费调用
  • API接口兼容Anthropic格式
  • CC Switch可视化配置
  • 支持Claude Code/Codex/OpenClaw
  • 128K上下文窗口

优点

  • 活动期间完全免费无限Token
  • Qwen3.6模型能力很强，35B参数够用
  • 支持128K上下文窗口
  • 可通过CC Switch接入Claude Code/Codex
  • 平台聚合多个主流模型

缺点

  • 免费活动6月底截止
  • 需要通过CC Switch配置使用
  • 暂不兼容Claude Code最新版本

使用建议

  • 先领权益，6月底前都免费
  • 用CC Switch配置比手动改配置文件简单
  • Claude Code如果有问题回退到v2.1.145版本
  • 高并发需求可联系讯飞工作人员额外领取

接入步骤

  1. 访问 maas.xfyun.cn 进入平台
  2. 在模型集市找到 Qwen3.6-35B-A3B 或 Qwen3.5-35B-A3B
  3. 点击「API调用」，创建授权应用，获取 modelId 和 APIKey
  4. 下载 CC Switch（开源工具）
  5. 添加供应商，填入API Key：
    • Claude Code请求地址: https://maas-api.cn-huabei-1.xf-yun.com/anthropic
    • Codex请求地址: https://maas-api.cn-huabei-1.xf-yun.com/v2
  6. 填入modelId和模型名称，测试通过后启用
