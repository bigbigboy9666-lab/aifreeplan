小米MiMo Code

小米基于OpenCode魔改的AI编程助手，开源MIT协议。最大的卖点是内置MiMo Auto通道，装完就能用，限时免费1个月，跑的是MiMo-V2.5模型，100万token上下文。实际跑下来编码能力不错，SWE-Bench成绩跟Claude Sonnet 4.6一个水平，但token效率高很多，同样的任务比Claude省40-60%的消耗。亮点是跨会话记忆、语音输入、自动从历史会话提取知识这些。免费期过了就得付费了，Token Plan最便宜的Lite档起步。

优点

  • ✓限时免费白嫖1个月，零配置直接用
  • ✓编码能力比肩Claude Sonnet 4.6
  • ✓token效率极高，比Claude省40-60%
  • ✓100万token超长上下文
  • ✓跨会话持久记忆，不会每次从头开始
  • ✓语音输入、自动知识提取等独家功能
  • ✓开源MIT，可以自己接别的模型

缺点

  • ✗免费只有1个月，过了得掏钱
  • ✗免费通道只给V2.5，Pro版要付费
  • ✗Credit消耗倍率高，宣传的16亿Credit实际用起来没那么多
  • ✗生态刚起步，社区还在建设中
  • ✗需要小米账号实名认证才能买套餐

核心功能

MiMo Auto零配置免费通道跨会话持久记忆（SQLite FTS5）Compose模式规范驱动开发语音输入（TenVAD + MiMo ASR）/dream自动知识提取 + /distill技能提炼Goal停止条件验证（防AI偷懒）子代理并行系统100万token上下文窗口开源MIT协议
