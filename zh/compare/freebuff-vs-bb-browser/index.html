Freebuff

免费AI编程Agent，终端里用自然语言改代码。基于Codebuff付费版，用广告换免费额度。默认用DeepSeek V4 Flash，支持Kimi K2.6、MiniMax M2.7等国产模型。

优点

  • ✓完全免费，广告换额度
  • ✓无需配置，开箱即用
  • ✓支持多个国产模型
  • ✓内置代码审查和浏览器操作
  • ✓开源MIT协议

缺点

  • ✗终端里有广告
  • ✗DeepSeek V4 Pro会用数据训练
  • ✗不是所有国家都可用
  • ✗需要联网，不能离线用

bb-browser

开源工具，"你的浏览器就是API"。CLI + MCP Server，让AI Agent能控制你的Chrome浏览器并使用你的登录状态。不需要单独配置API Key，直接复用你浏览器里已登录的账号。适合需要AI帮你操作网页、自动化浏览器任务的开发者。

优点

  • ✓开源免费，MIT协议
  • ✓复用浏览器登录状态，不需要额外配置
  • ✓支持CLI和MCP Server两种模式
  • ✓让AI Agent能操作真实网页

缺点

  • ✗需要本地运行Chrome
  • ✗依赖Node.js环境
  • ✗项目较新，社区生态还在建设

核心功能

CLI命令行工具MCP Server集成Chrome浏览器控制复用登录状态AI Agent自动化
