    Sonnet 模型的编程能力

    Claude Sonnet 在编程领域表现非常强。多个基准测试显示：

  • SWE-Bench（代码修复）：Sonnet 排名前列
  • HumanEval（代码生成）：接近满分
  • 实际体感：写代码、调试、代码审查都很靠谱

    • 如果你用 Claude 做编程，建议配合 Cline（开源 AI 编程助手）。Cline 可以把 Claude 的编程能力直接嵌入到 VS Code 里，体验比网页版好很多。

    ---

    Artifacts：免费版的隐藏功能

    Artifacts 是 Claude 的一个特色功能。当你让 Claude 写代码或生成内容时，它会在右侧显示一个可预览的面板：

  • 写 HTML/CSS → 直接预览网页效果
  • 写 Python → 直接运行并显示结果
  • 生成图表 → 直接可视化
  • 写 SVG → 直接渲染图形

    • 这个功能免费版就能用。对于快速原型、学习编程、生成可视化内容特别有用。

    Claude Pro 值不值 $20/月

    $20/月换来的是：

  • Sonnet 5 倍用量（从每天 20-30 条到 100+ 条）
  • Opus 模型访问（最强推理能力）
  • Claude Code（终端 AI 编程工具）
  • Projects（项目管理，可以设定项目上下文）

值不值取决于你的使用场景。如果你每天重度使用 Claude（超过 1 小时），$20/月合理。如果只是偶尔问个问题，免费版够了。

能。支持 PDF、代码文件、文档等。200K 上下文意味着你可以上传非常长的文件。
