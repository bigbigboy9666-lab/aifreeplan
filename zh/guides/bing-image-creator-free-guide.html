Bing Image Creator 免费额度详解

免费版的核心数据：

  • 每天100次加速生成（Boosted）
  • 超过100次后降速但不收费
  • 分辨率 1024x1024
  • 无水印
  • 不需要绑信用卡
  • 个人使用免费

"加速生成"和"普通生成"的区别在于等待时间。加速生成通常10-30秒出图，普通生成可能要排队几分钟甚至更久。但不管哪种，都是免费的。

100次加速额度每天重置，用完了就变成慢速队列，第二天又恢复100次加速。

怎么用 Bing Image Creator

三步搞定：

  1. 打开 bing.com/images/create
  2. 用 Microsoft 账号登录（没有就注册一个，免费的）
  3. 输入英文描述，点"Create"

等10-30秒，会生成4张图片供你选择。不满意可以修改描述重新生成。

重要提示：DALL·E 对英文的理解比中文好很多。用英文写提示词效果明显更好。比如写"a cat sitting on a windowsill watching the sunset, watercolor style"比写"一只猫坐在窗台上看日落"效果好。

Bing Image Creator 和其他免费 AI 绘画工具对比

Bing Image Creator 的主要竞品：

Ideogram：每天25次免费生成，文字渲染最好 Leonardo AI：每天15张免费，有高级控制选项 Playground AI：每天500张免费，但画质一般 Adobe Firefly：每月25积分，Adobe生态整合好

Bing Image Creator 的优势是完全免费、DALL·E画质、不用翻墙。劣势是风格控制不如专业工具灵活，排队时间有时候比较长。

提示词技巧：怎么让 Bing Image Creator 出好图

提示词写法直接影响出图质量。几个实用技巧：

加风格描述：在描述后面加上"oil painting"、"watercolor"、"digital art"、"photorealistic"等风格词。

加构图描述：加上"in the style of Studio Ghibli"、"cinematic lighting"、"close-up shot"等。

加细节：颜色、材质、光线、背景都要写具体。"A red sports car on a rainy Tokyo street at night, neon reflections"比"a red car"好10倍。

避免抽象概念：DALL·E 不擅长抽象概念，越具体的描述出图越好。

❓ 常见问题

Bing Image Creator 完全免费吗？

是的，完全免费。每天100次加速生成，超过后降速但仍免费。不需要绑信用卡。

Bing Image Creator 用的什么模型？

用的是 OpenAI 的 DALL·E 模型，这是目前唯一免费用 DALL·E 的官方渠道。

Bing Image Creator 支持中文提示词吗？

支持，但效果不如英文。DALL·E 对英文的理解明显更好，建议用英文写提示词。

Bing Image Creator 生成的图片能商用吗？

免费版仅供个人使用，商用需要额外授权。

Bing Image Creator 和 Ideogram 哪个好？

Bing 完全免费每天100次，Ideogram 每天25次但文字渲染更好。如果需要图片中有文字，选 Ideogram；纯绘画选 Bing。

Bing Image Creator 需要翻墙吗？

在中国大陆需要。但在大部分地区可以直接访问，用 Microsoft 账号登录即可。
