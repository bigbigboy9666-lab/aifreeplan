我的失败教训：API调用踩过的坑

坑1：OpenAI的$5很快就用完了

我以为$5能用很久，结果用GPT-4o跑了一个数据分析任务就花了$3。GPT-4o mini便宜很多，日常用mini就行。

坑2：Gemini API需要科学上网

在国内直接调用Gemini API会超时。需要配置代理或者用Google AI Studio的网页版。

坑3：免费API都有速率限制

DeepSeek每分钟60次，Gemini每分钟15次。做批量处理时很容易触发限制。建议加延时或用异步调用。

坑4：高峰期DeepSeek排队严重

工作日早上9-11点是高峰期，API响应可能要等10-30秒。非高峰期（凌晨或周末）速度很快。

坑5：免费额度不累积

大部分平台的免费额度每天重置，用不完就浪费了。建议每天规划好用量，不要囤积。

常见问题

免费API能商用吗？

大部分可以。DeepSeek、Gemini、OpenAI的免费额度都允许商用，但需遵守服务条款。建议查看各平台的使用政策。

怎么开始调用免费API？

三步：1) 注册账号获取API Key；2) 用curl或Python发送请求；3) 处理返回结果。DeepSeek和Gemini的文档最友好。

哪个免费API最适合新手？

DeepSeek API。文档最全、额度最大、兼容OpenAI接口格式（已有OpenAI代码只需改base_url）。

免费API有并发限制吗？

有。DeepSeek每分钟60次，Gemini每分钟15次，OpenAI根据tier不同。做批量任务时注意控制并发。

2026年下半年免费API额度会变吗？

大概率会收紧。AI行业竞争格局趋稳后，各家会逐步减少免费额度。建议趁现在额度充裕时多用。

免费API和付费API差距大吗？

主要差在速率限制和模型选择。免费额度足够个人使用和开发测试，但高频商业应用需要付费。

常见问题

哪个平台免费模型最多？

NVIDIA NIM有80+免费模型，ModelScope有40+，DmxAPI有20+。NVIDIA数量最多但响应较慢，ModelScope多模态最强。

哪些平台需要充值才能用？

AIHubMix需充1美金、302.AI和CherryIn需充1美元、OpenRouter充$10可提升到1000次/天。DmxAPI充1元激活全部免费模型。

免费API能用于生产环境吗？

不建议。免费API都有速率限制，高峰期429限流是常态。做demo和学习可以，对外服务建议付费。

想用GPT-4o应该选哪个？

GitHub Models可以免费用gpt-4o，15 RPM / 150 RPD，无需信用卡。

快下架了。需要用旧版控制台的apikey，新版没有hunyuan-lite模型。建议尽早迁移到其他平台。
