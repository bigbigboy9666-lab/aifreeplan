MiMo Code 是什么

MiMo Code 是小米大模型团队基于 OpenCode 框架开发的终端 AI 编程助手，采用 MIT 协议开源。与 Cursor、Windsurf 等 IDE 不同，MiMo Code 运行在终端中，适合习惯命令行工作流的开发者。

核心亮点：

  • 内置免费模型：MiMo-V2.5 模型限时免费，无需额外配置 API Key 即可使用
  • 持久记忆系统：通过独立 subagent 自动保存会话状态，解决长对话「健忘」问题
  • Compose 模式：一键完成「设计 → 规划 → 编码 → 测试」全流程
  • Dream 命令：7 天周期记忆沉淀，AI 越用越懂你的代码风格
  • 语音输入：支持语音描述需求，不用打字
  • 全面中文汉化：界面、文档、错误提示全部中文化

安装与快速上手

系统要求

  • 操作系统：macOS、Linux、Windows（WSL）
  • Node.js 18+ 或通过 npx 直接运行
  • 终端环境（iTerm2、Terminal、Windows Terminal 等）

安装步骤

最简单的安装方式是通过 npm：

npm install -g @anthropic-ai/mimo-code

或者直接用 npx 运行（无需安装）：

npx @anthropic-ai/mimo-code

首次启动后，MiMo Code 会引导你完成配置。选择使用内置的 MiMo-V2.5 模型即可免费开始使用，无需注册任何第三方 API。

核心功能详解

1. 持久记忆系统

传统 AI 编程助手的最大痛点是「健忘」——对话超过 20 轮就开始丢失上下文。MiMo Code 的解决方案是通过独立的 subagent 自动管理记忆：

  • 每次会话的关键决策、代码变更、项目结构信息会被自动保存
  • 新会话启动时自动加载历史记忆，无需重复描述项目背景
  • 记忆按项目隔离，不同项目之间互不干扰

2. Compose 模式

Compose 模式是 MiMo Code 的核心差异化功能。输入一个需求描述后，它会自动执行四个阶段：

  1. 设计（Design）：分析需求，生成技术方案和架构设计
  2. 规划（Plan）：拆解任务为可执行的步骤列表
  3. 编码（Code）：按步骤逐步实现代码
  4. 测试（Test）：自动编写测试用例并验证

整个过程无需人工干预，适合快速原型开发和功能迭代。

3. Dream 命令

输入 /dream 命令，MiMo Code 会对过去 7 天的交互进行记忆沉淀，提取你的编码习惯、常用框架、代码风格偏好等信息。下次对话时，AI 会自动参考这些偏好，生成更符合你风格的代码。

4. 语音输入

不想打字？MiMo Code 支持语音输入功能。按住快捷键说出你的需求，AI 会自动转录并执行。这个功能在描述复杂需求时特别方便。

5. 第三方模型接入

除了内置的 MiMo-V2.5，MiMo Code 还支持接入其他模型 API：

  • DeepSeek：DeepSeek-V3 / DeepSeek-Coder，有免费额度
  • Qwen：通义千问系列，阿里云提供免费 API 额度
  • GLM：智谱 GLM-4/5 系列，新用户送 500 万 token
  • OpenAI：GPT-4o 等模型（付费）
  • Anthropic：Claude 系列模型（付费）
