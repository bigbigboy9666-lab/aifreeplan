数据来自 GitHub 官方文档、Microsoft Learn 和第三方评测，2026 年 5 月实测。

先说结论

去年帮学弟申请 GitHub Education 的时候发现，GitHub Copilot 免费版每月给 2000 次代码补全，学生和开源维护者还能白嫖 Pro 版——不限次数加 Agent 模式。如果你是学生，这是目前最值的 AI 编程福利，没有之一。

免费版到底给什么

关键：学生和开源项目维护者可以免费拿到 Pro 版。GitHub Education 认证流程很简单，有 .edu 邮箱就行。

竞品对比——免费的含金量

学生怎么白嫖 Pro 版

GitHub Education 认证三步走：第一步，去 education.github.com 用 GitHub 账号登录选"学生"身份；第二步，验证学生身份——用 .edu 邮箱接收验证码最快（1 分钟搞定），没有的话上传学生证照片（1-3 天审核）；第三步，认证通过后自动获得 Copilot Pro 免费、GitHub Pro 免费和一堆其他开发者工具的免费额度。注意每年需要重新验证一次，毕业后就不能续了。

开源维护者怎么免费

如果你是 GitHub 上公开项目的维护者，也可以申请 Copilot Pro 免费。条件是项目公开、你是项目的 collaborator 或 owner、项目有一定活跃度（不是空仓库）。去 github.com/settings/copilot 申请选"开源维护者"身份就行。

免费版够用吗

2000 次补全/月大概是什么概念？如果你每天写 2 小时代码，大概每天用 100-150 次补全，一个月 2000 次只够用半个月左右。所以免费版适合偶尔写代码的人、刚开始学编程的新手和想体验 AI 编程再决定要不要付费的人。如果你是每天写 8 小时的职业开发者，2000 次大概一周就用完了，建议直接上 Pro 版或者用 Codeium（免费无限补全）。

Agent 模式：Pro 版的杀手功能

Agent 模式是 Copilot 最大的差异化功能——你给它一个任务描述，它会自己分析代码库结构、制定修改计划、自动修改多个文件、运行测试验证，如果出错还会自动修复。这个功能只有 Pro 版有，免费版只能用 Chat 模式（一问一答，不支持自主编码）。如果你用过 Cursor 的 Composer 模式或者 Windsurf 的 Cascade 模式，Copilot Agent 是同类功能，区别是 Copilot 直接在 VS Code 里用，不需要换编辑器。

❓ 常见问题

GitHub Copilot 免费版有次数限制吗？

有。每月 2000 次代码补全，50 次 Chat 消息。用完就没了，下个月重置。

学生认证需要什么？

.edu 邮箱最快，1 分钟搞定。没有的话上传学生证，1-3 天审核。

免费版支持哪些 IDE？

VS Code 和 JetBrains 全系列（IntelliJ、PyCharm、WebStorm 等）。

Copilot 和 ChatGPT 写代码哪个好？

Copilot 是 IDE 内嵌的，写代码时实时补全，不用切换窗口。ChatGPT 更适合问问题和讨论方案。

毕业了还能用免费 Pro 吗？

不能。每年需要重新验证学生身份，毕业后会降回免费版。
