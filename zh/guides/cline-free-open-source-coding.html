)

Cline 是一个开源 AI 编程助手，可以在 VS Code、JetBrains 和命令行里使用。和 GitHub Copilot、Cursor 这些商业工具不同，Cline 有几个核心区别：

完全开源：代码在 GitHub 上，Apache 2.0 协议，可以商用、修改、分发。 不锁定模型：你可以接任何 AI 模型——Claude、GPT-4o、Gemini、DeepSeek、Qwen，通过 OpenRouter 可以用 200+ 模型。 自备 API Key：没有订阅费，你只需要付 API 调用费用。用 DeepSeek 的话，编程任务每小时大概 $0.01-0.05。 社区驱动：GitHub 上星标破万，更新频率高，社区非常活跃。

Plan/Act 双模式

Cline 有两个工作模式，这是它最聪明的设计：

Plan 模式：

  • Cline 先分析你的需求
  • 制定修改计划
  • 列出要改哪些文件、怎么改
  • 你确认后才执行

Act 模式：

  • Cline 直接执行任务
  • 自动创建/修改文件
  • 运行终端命令
  • 自动修复错误

建议：先用 Plan 模式让 Cline 制定计划，确认没问题后切到 Act 模式执行。这样更可控，不容易出意外。

Kanban 多任务并行

Cline 有一个独特的 Kanban 看板功能。你可以同时启动多个编程任务，Cline 会在后台并行处理。比如：

  • 任务 1：重构用户模块
  • 任务 2：修复登录 bug
  • 任务 3：写单元测试

三个任务同时跑，你可以在看板上看到每个任务的进度。这对大型项目特别有用。

❓ 常见问题

Cline 真的完全免费吗？

是的。Cline 本身完全免费、开源、无限制。你需要自备 AI 模型的 API Key，这部分有成本。

Cline 和 GitHub Copilot 哪个好？

Cline 免费无限制但需要自备 Key，Copilot 免费版有 2000 次限制。Cline 更划算。

Cline 支持哪些 AI 模型？

通过 OpenRouter 可以用 200+ 模型，包括 Claude、GPT、Gemini、DeepSeek、Qwen 等。

Cline 安全吗？代码会泄露吗？

Cline 本身是开源的，代码可审计。安全性取决于你选择的 API 提供商。

新手能用 Cline 吗？

能，但需要一点配置（安装扩展 + 填 API Key）。如果不想配置，用 Copilot 免费版更简单。
