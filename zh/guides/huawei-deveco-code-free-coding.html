华为DevEco Code免费攻略：GLM-5.1不限量白嫖指南

更新时间：2026年6月15日

白嫖AI编程助手的名单上，又多了一个重量级玩家——华为。华为最近发布了一个叫DevEco Code的CLI工具，跟之前那些送Token的玩法不太一样。它内置了智谱的GLM-5.1模型，完全免费，不限额度、不限次数。装好就能用，不需要抢购，不需要排期。

GLM-5.1是什么水平

GLM-5.1是智谱今年3月发布的旗舰开源模型，在Claude Code编码评测中得了45.3分，达到Claude Opus 4.6（47.9分）的94.6%，比前代GLM-5提升了28%。而且这个模型是完全基于华为昇腾芯片训练的——也就是说，华为跟智谱在这个模型上本来就有深度合作。

现在华为把GLM-5.1作为DevEco Code的内置模型免费开放，等于把自家芯片训练的模型，通过自家工具，免费提供给开发者。逻辑上是通的，对开发者来说是实打实的福利。

三分钟上手

使用门槛很低，不需要鸿蒙开发经验，也不需要安装DevEco Studio。

第一步，装工具。

npm install -g @deveco/deveco-code

装完跑deveco -v，看到版本号就说明成功了。

第二步，登录。

终端输入deveco，选择「自动跳转到浏览器登录」，用华为账号登录就行。没有账号可以注册，免费的。

第三步，开用。

登录完成后直接输入需求，默认就是GLM-5.1模型，无需手动选择。整个过程三分钟搞定，比配置大部分AI编程工具都快。

特色功能：皮肤主题

DevEco Code内置多种编辑器皮肤主题，支持深色/浅色模式切换，界面美观，编码体验舒适。截图中展示的就是默认的深色主题，视觉效果很棒。

跟其他免费方案有什么不同

市面上的免费AI编程工具很多，但各有各的短板。

  • 硅基流动和讯飞星辰送Token，但有用量上限和速率限制
  • 智谱自己的Coding Plan需要每天早上抢，抢到了才有额度
  • MoMA送2500万Token，但也需要实名认证且有有效期

DevEco Code这个方案的不同之处在于：它不限额度。你不用担心Token用完、不用排队抢购、不用算着用量过日子。只要华为还在运营这个工具，GLM-5.1就能免费调用。

当然，这个工具目前定位是辅助鸿蒙开发的，内置GLM-5.1只是其中一项能力。但实测下来，不干鸿蒙相关的活，单纯拿它当通用编程助手用也完全没问题。

一个问题：能接入其他工具吗

DevEco Code是一个独立的CLI工具，有自己独立的终端界面，不像其他API那样可以自由接入Cursor、Claude Code、Windsurf等第三方工具。这是它跟OpenAI兼容API的本质区别。

目前看来，这个工具更适合把它当成一个独立的AI编程环境来用，而不是作为其他工具的模型后端。如果你想在Cursor或者Claude Code里用GLM-5.1，还是得走智谱官方的API或者Coding Plan。

适合谁用

  • 如果你在找免费的AI编程助手，又不想折腾API Key和用量管理，DevEco Code几乎是最省心的选择。装好登录就能用，不需要任何配置。
  • 如果你是鸿蒙开发者就更赚了——DevEco Code本身就是为鸿蒙应用开发设计的CLI工具，GLM-5.1是锦上添花。
  • 如果你已经在用其他AI编程工具（比如Claude Code或Codex），而且用得很好，那DevEco Code可能不会取代你现有的工作流。但它作为一个备选方案——特别是主力工具出问题的时候——价值很大。

观察者点评

华为这一手挺聪明的。GLM-5.1本身就是基于昇腾芯片训练的，把自家芯片训练的模型放到自家开发工具里免费提供，既能吸引开发者使用DevEco Code，又能展示昇腾生态的AI能力，一举两得。

对开发者来说，不管华为的动机是什么，结果就是多了一个不限量免费的编程助手可用。GLM-5.1虽然不是最强的模型，但45.3的编码评分已经接近Claude Opus 4.6的水平，日常开发完全够用。考虑到智谱的Coding Plan还要每天抢，华为这个「无需抢购、不限额度」的方案，对普通开发者来说反而更实用了。
