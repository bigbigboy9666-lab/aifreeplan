开源AI编程助手，支持VS Code、JetBrains和命令行。不限定模型，可以对接Claude、GPT、Gemini等200+模型。免费使用，自带API Key就行。社区非常活跃，GitHub上星标破万。

Cline 主要功能

  • VS Code 扩展
  • JetBrains 插件
  • CLI 命令行模式
  • Kanban 多任务并行
  • Plan/Act 双模式
  • 支持 200+ AI 模型
  • MCP 服务器扩展
  • 自动修复 Linter 错误

优点

  • 完全开源免费，Apache 2.0 协议可商用
  • 不锁定模型，自由选择最合适的
  • 社区活跃，更新快
  • 支持终端、编辑器、Web 多端
  • 自动检查代码错误并修复

缺点

  • 需要自备 API Key，新手门槛稍高
  • 复杂任务消耗 Token 快，需要关注账单
  • 配置项多，初次上手需要学习

使用建议

  • 新手建议先用 Claude Sonnet 模型，性价比最高
  • 在 .clinerules 文件里定义项目规则，效果提升明显
  • 搭配 OpenRouter 可以用 200+ 模型，价格透明
