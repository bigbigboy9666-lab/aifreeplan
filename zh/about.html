我们关心什么

我们只关心一件事：免费额度。不是功能列表，不是技术参数，就是"不花钱能用多少"这个最实际的问题。

  • 注册送多少额度/credits
  • 每天/每月有没有使用上限
  • 免费版带不带水印
  • 免费版能不能商用
  • 额度多久重置一次
  • 有没有隐藏的付费门槛

这些信息我们会持续跟踪更新。AI 工具的免费政策变化很快——今天无限免费，明天可能就要收费了。我们尽量在第一时间同步这些变化。

收录标准

我们不收录纯付费工具。一个工具如果完全没有免费使用的途径，它不会出现在 AIFreePlan 上。我们的收录标准是：至少满足以下条件之一：

  • 有永久免费版或免费套餐
  • 新注册用户赠送可使用的额度
  • 有限时免费活动或试用期
  • 通过特定渠道（如教育优惠、开源计划）可免费使用

数据来源

我们的数据来自以下几个渠道：

  • 各工具官方网站的定价页面和文档
  • 我们自己注册试用后的实测数据
  • 社区用户（包括 GitHub Issues 留言板）的反馈和纠错
  • 各工具的官方社交媒体公告

如果你发现某个工具的额度信息有误或过时，欢迎通过 GitHub Issues 告诉我们，我们会尽快核实和修正。
