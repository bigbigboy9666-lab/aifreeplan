Poe

★★★★☆ 每日积分（模型不同消耗不同）/天 付费起步: $20/month

访问官网 →

Quora 做的 AI 聚合平台，一个入口用所有主流模型。免费用户每天有一定积分，可以切换 GPT-4o、Claude Sonnet、Gemini、Llama 等模型。付费 $20/月解锁更多用量和高级模型。最大优势是不用开一堆网站。

免费额度详情

  • 额度: 每日积分（模型不同消耗不同）
  • 重置: 每天重置
  • 水印: 无
  • 商用: 付费后可商用

主要功能

  • 多模型聚合（GPT-4o、Claude、Gemini等）
  • 自定义 Bot 创建
  • Bot 市场（社区创建的 Bot）
  • 文件上传分析
  • 图片生成（集成 DALL-E、SD等）
  • API 访问
  • iOS/Android App

优点

  • 一个平台用所有主流 AI 模型
  • 免费额度够日常使用
  • 自定义 Bot 和社区 Bot 生态丰富
  • 不用注册一堆账号
  • 有 API 可用

缺点

  • 免费积分有限，重度使用不够
  • 付费版 $20/月和直接订阅各家差不多
  • 高级模型（如 Opus）消耗积分多
  • 国内访问需要特殊网络

使用建议

  • 想同时试多个模型的用户首选 Poe
  • 免费积分省着用，日常小问题用免费模型就行
  • 社区 Bot 很多好用的，搜搜看有没有你需要的场景
